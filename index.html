FAQ'S

Is Desktop for me?
If you want to share your web app as a downloadable desktop app, then ToDesktop is perfect for you.
Do I need to code in Electron?
No, ToDesktop handles all the Electron code for you. Just package your web app.
Is there offline support?
Yes! Your app can cache pages and run offline using service workers.
Can I push updates?
Yes, with ToDesktop, you can push automatic app updates to users.
Which platforms are supported?
Windows, macOS, and Linux are supported out of the box.
Can I customize the app UI?
Yes, you can customize the title bar, splash screen, and icons easily.
